# Claude Theme for Zed

A minimal Zed theme inspired by Claude.

Clean syntax colors, soft contrast, and a calm interface designed for focused coding.

Supports both light and dark modes.

---

## Themes

* Claude Dark
* Claude Light

---

## Installation

Copy the theme file into:

```bash
~/.config/zed/themes
```

Then open Zed:

```text
Cmd/Ctrl + Shift + P
→ Theme Selector: Toggle
```

Select:

* Claude Dark
* Claude Light

---

## Philosophy

This theme is built around:

* low visual noise
* readable contrast
* restrained colors
* calm editing experience

Designed for people who prefer quiet interfaces over flashy ones.
