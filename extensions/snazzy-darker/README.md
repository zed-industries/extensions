# Snazzy Darker for Zed

A darker, high-contrast variant of the popular Snazzy color scheme, optimized for readability and reduced eye strain during long coding sessions.

## Features
- Full dark mode support
- Carefully balanced contrast ratios for all UI elements
- Complete syntax highlighting for all major programming languages
- Custom terminal ANSI color palette matching the Snazzy theme
- Optimized for both code and UI readability

## Installation

### Local Dev Installation
1. Open Zed
2. Go to `Settings > Extensions`
3. Click `Install Dev Extension`
4. Select the `snazzy-darker` directory
5. Activate the theme from the theme picker

### Official Installation (once published)
1. Open Zed
2. Go to `Settings > Extensions`
3. Search for "Snazzy Darker"
4. Click Install
5. Select the theme from the theme selector
